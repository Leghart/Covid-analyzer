Covid Analyzer

General information

Welcome to my project of analyzing Covid in Poland. I downloaded raw data from here: link and then, I use machine learning to predict a new cases for future. Currently it's simple version of project so I can only predict one day. That site is divided into 4 main sections, the first is what you are reading right now, second section presents a table of last 10 records I collected. There are several columns that contain the most important information for example: new cases of infection, new deaths, total number of cases, total number of deaths, etc. The third section contains graphs of data with prediction which I made. It focuses the most important points that should be predicted. The last section will tell you what could will happen tomorrow. This page is updated automatically at 6 p.m. every day.
